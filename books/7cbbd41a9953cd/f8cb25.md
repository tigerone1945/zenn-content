# 付録：Udemy講座・Kindleシリーズのご案内

このBookは、AIエージェントの考え方・設計方法・全体像を理解することを目的に執筆しています。

一方で、実際の業務でAIエージェントを活用するためには、

* 設計を理解する
* 実際に構築する
* 運用できるようになる

という3つのステップが必要です。

そのため、本Bookと連携する形で、

* Kindleシリーズ（体系的な知識習得）
* Udemy講座シリーズ（実践構築）

を順次公開しています。

Bookで全体像を理解し、Kindleで知識を深め、Udemyで実際に構築することで、AIエージェント活用を体系的に学べる構成になっています。

---

# 本BookとUdemy講座の対応

本BookではAIエージェントの考え方と設計方法を学びます。

Udemy講座では、その考え方を実際の業務システムとして構築する方法を学びます。

| 本Book | 内容               | Udemy講座  |
| ----- | ---------------- | -------- |
| 第1〜4章 | AIエージェントの基礎・設計思想 | 全講座共通    |
| 第5章   | 商品レビュー感情分析システム   | 講座1で発展実装 |
| 第6章   | 問い合わせ分類AI        | 講座2で発展実装 |
| 第7章   | 自律型メール秘書         | 講座2で発展実装 |
| 第8章   | 議事録生成システム        | 講座3で発展実装 |
| 第9章   | 文書DXシステム         | 講座3で発展実装 |

Bookでは「なぜそう設計するのか」を学びます。

Udemyでは「実際にどう構築するのか」を学びます。
